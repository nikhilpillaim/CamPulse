#  CamPulse: Professional Collaboration Hub

**CamPulse** is a high-fidelity UI/UX prototype designed to streamline project collaboration and hackathon team formation. It provides a structured environment for users to find teammates, discuss ideas, and share protected project data.

### 🚀 Key UI Modules
* **Three-Column Dashboard:** A professional spatial layout for efficient navigation between trending topics, house threads, and event bulletins.
* **Ghost vs. Resident System:** A role-based access model where "Ghosts" can explore the community, while "Verified Residents" gain full interaction and voting privileges.
* **Salted Content:** A data-protection feature that uses CSS-blur to hide sensitive meeting details or project logic from unverified users.
* **"Make Room" Module:** A dedicated interface for initiating private peer-to-peer collaboration requests.

### 🛠️ Tech Stack
* **Frontend:** HTML5 & Tailwind CSS (for modern, responsive UI components).
* **State Management:** Alpine.js (used for reactive, backend-free page switching and role logic).
* **Design Principles:** Glassmorphism, F-Pattern scanning, and Spatial UI.

### 📖 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Use the "Enter as Ghost" or "Join as Resident" buttons to test the different UI states.
